## What it includes?

Universe superclass:

- Server class
- User class
   1. Administrator subclass
- Message class
   1. Text messages subclass
   2. Voice messages subclass
   3. Photo messages subclass
- Settings class
- Chat History class
- Achievements class
   1. Roles subclass
   2. Gifts subclass
